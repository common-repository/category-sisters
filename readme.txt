Add a widget to list category sisters on a Category/post page. (useful when you have a hierarchy in categories) 

== Description ==

If you have categories with some hierarchy, widget generated by the plugin, add, on a category or a post page, a list of categories from the same parent. 

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to widgets area, Widget 'Category Sisters' can be used in sidebar
